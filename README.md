# Zig Cheatsheet

A little cheatsheet i've made whilst learning Zig. I'll try to keep updating this repo every breaking Zig update so it's up-to-date.

It is recommended to use a Zig LSP such as ZLS, Zigscient, or others, as they provide inlay hints — the codebase is written compactly and relies on inlay hints to convey types.

This cheatsheet won't include how `zig test` works, testing in general, the build system, C interop, and more. i want to keep this repo minimal and language-specific. Maybe if i have the time to add them, i will.
